# PR Semantic Search

An experimental RAG-based semantic retrieval system designed for automotive PR knowledge management.

## 🎯 Project Goal

Build a controllable Retrieval-Augmented Generation (RAG) system to:

- Reduce hallucination
- Improve evidence transparency
- Provide structured PR strategy outputs

This project focuses on retrieval stability, ranking logic, and context control rather than feature expansion.

## 🧱 Architecture

- FAISS vector index
- Custom chunking strategy
- Controlled evidence injection
- LLM summarization layer (DeepSeek / OpenAI compatible)

## 🔬 Current Focus

- Hallucination diagnosis
- Retrieval ranking optimization
- Context size control
- Deduplication logic

## 🚀 Roadmap

- Evidence scoring transparency
- Structured output enforcement
- Retrieval evaluation metrics
- Modular pipeline refactor
